Status: review

> 后端为主：系统自动类里程碑订阅既有领域事件 → 幂等标记完成。**护栏**：只用
> `@TransactionalEventListener`(AFTER_COMMIT) + `@Async` 订阅既有领域事件 + 定时类走 `@Scheduled`，

## Story
As a **用户**, I want **我做了对应的事系统就自动点亮里程碑**, so that **无需手动即可解锁系统自动类节点**。
