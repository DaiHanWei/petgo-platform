# Story 9.3: 兽医咨询订单只读管理

Status: review

<!-- Note: Validation is optional. Run validate-create-story for quality check before dev-story. -->

> **所属**：V1.1 Epic 9 第三个 Story（**后端 + admin 模板**）。承接 9-1 权限码 `order.view`/`order.export`。交付：运营**只读**查看兽医咨询订单全流程——列表 + 展示态 + 重播计数 + 阶段事件时间线 + **待核查标记**（人工，无自动拦截）+ CSV 导出。**无退款入口**（退款走客服工单 4-x，AB-8B 铁律）。
> **关键事实**：`consult_orders`（V66）已有成交快照（amount/vet_payout/vet_share_rate/unit_price/paid_at/session times/refund_rejected）+ `consult_order_stage_events`（append-only 阶段流 ACCEPTED/PAID/SESSION_STARTED/SESSION_ENDED/REFUND_*）。**展示态复用** `OrderCenterService.vetStatusCode` 同款派生（IN_PROGRESS / COMPLETED / COMPLETED_REFUND_REJECTED / REFUNDING / REFUNDED）。
> **重播计数**：源在 `consult_requests.rebroadcast_count`，但**建单即删 request**（A-5 claim-first）→ 订单不留。故本 story **建单时快照** `rebroadcast_count` 到 `consult_orders`（V79 加列 + `ConsultPayService.buildOrderAndConvert` 一行快照，**不改 `ConsultOrder.inProgress` 工厂签名**避免 13 处测试涟漪，用 managed 实体 setter + 提交 flush）。
> **待核查标记**：V79 加 `admin_verify_status`(NULL/`TO_VERIFY`/`VERIFIED`) + note/by/at。标记是**写**操作，接 `AdminAuditService` 审计；门控 `order.view`（内部注记，低危）。

## Story

As a **运营**,
I want **只读查看兽医咨询订单全流程**,
so that **我能追踪订单、标记待核查（AB-8B）**。

## Acceptance Criteria

> **验证层**：**L0**（展示态派生/CSV 组装/校验单测、`mvn compile`）· **L1**（列表/详情读、重播快照、标记落库 + 审计、导出、schema validate）。无 L2。

### AC1 — V79 加列（重播快照 + 待核查）+ schema validate

**Given** `consult_orders` ALTER 加 `rebroadcast_count`(default 0) + `admin_verify_status`(NULL/TO_VERIFY/VERIFIED CHECK) + `admin_verify_note`/`admin_verify_by`/`admin_verify_at`（**Flyway V79**）
**When** 上下文启动 + 支付建单
**Then** `ddl-auto=validate` 一致；建单时 `rebroadcast_count` = 该 request 的重播次数快照
> 验证层：**L0**（迁移 DDL + 实体编译）+ **L1**（建单后订单 rebroadcast_count 正确 + validate 过）。

### AC2 — 只读列表 + 展示态 + 阶段时间线，无退款入口

**Given** `order.view` 门控 + 订单列表（倒序，分页）+ 详情（成交快照 + 展示态 + 阶段事件时间线）
**When** 运营查看
**Then** 只读呈现（展示态复用 vetStatusCode）；**页面无任何退款按钮/入口**（退款走工单）；PII 按既有脱敏
> 验证层：**L1**（列表返回订单 + 状态码正确；详情含阶段事件；无退款 action）+ **L0**（展示态派生单测覆盖 5 码含 refund_rejected）。

### AC3 — 待核查标记（人工，审计）

**Given** `order.view` + 标记动作
**When** 标记 TO_VERIFY / VERIFIED（可带备注）
**Then** 落 `admin_verify_status`/note/by/at + `AdminAuditService.record` 哈希链；**无自动拦截**（纯标记，不改订单状态/不触发退款）
> 验证层：**L1**（标记落库 + 审计 1 条 + 订单业务状态不变）+ **L0**（非法状态值拒）。

### AC4 — CSV 导出

**Given** `order.export` 门控
**When** 导出
**Then** 返回 CSV（订单号/用户/兽医/金额/分成/状态/重播/待核查/时间），Content-Type text/csv + 下载头；PII 脱敏一致
> 验证层：**L1**（导出 200 + text/csv + 含表头与行）+ **L0**（CSV 行组装 + 转义单测）。

## Dev Notes

- **Flyway 号权威**：V79（当前 max V78=9-2）。决策 E2 顺延。
- **无退款入口铁律**（AB-8B）：本页只读 + 标记，绝不放退款按钮（退款只走客服工单两段审批 4-3/4-6）。
