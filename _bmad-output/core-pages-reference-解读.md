## 1. 这是什么

一份**用单文件 HTML 渲染的高保真页面原型集**，把 TailTopia App 的核心页面以「手机外壳 + 右侧图例/导航面板」的形式静态呈现，可在浏览器里点导航按钮切换页面、看动效。它**不是设计规范文档，也不是代码**，而是给人「看效果、对齐视觉与交互」用的参考稿。

文件自带的元数据（右侧「文件说明」面板）：

| 项 | 值 |
|---|---|
| 版本 | V1.0.0 · 2026-06-15 |
| 字体 | Poppins |
| 主色 | `#845EC9`（violet-500） |
| **声明的技术栈** | **React Native** |
| 图标库 | phosphor-react-native |
| 自定义「核心页面」 | 首页 Feed · 成长档案 · 问诊 · 我的 · 内容发布（5 个 Tab） |

> ⚠️ 元数据里写的是 **React Native + phosphor-react-native**，与本项目实际栈（**Flutter / Riverpod**）不一致。这只是原型作者标注的目标栈，落地时按本仓库 Flutter 为准——记录在此供后续对齐用，不在本文展开。

### 设计语言（右侧图例面板提取）

- **配色**：
  - `#845EC9` violet-500 — 主色（用户端）
  - `#9E83DA` violet-400 — 辅助色
  - `#F0425A` red-500 — **Pop Art 错位色 / 红色危险态**
  - `#544864` ink-700 — 文字
  - `#F8F2FF` violet-50 — 仅做局部点缀（画布背景须纯白，不能带紫调）
  - `#FFFFFF` — 卡片 Surface
  - 兽医端专用：`#5BCBBB`（薄荷绿）+ 深色顶栏 `#2B2540`
- **分诊三色**：🟢 正常 / 🟡 需咨询(`#F6A609`) / 🔴 紧急(`#F0425A`)
- **字阶**：Display 26/700 · Heading 19/600 · Body 15/500 · Body 13/400 · Caption 11/500 全大写标签
- **圆角**：签名圆角 **12px**（按钮/搜索框/输入框），另有 4/8/16/32/full
- **Pop Art 错位**：Tab 激活态下层红 `#F0425A` 向右下偏移 +3px、上层紫 `#845EC9` 原位，露出红色「套色偏位」边；过渡 150ms，仅动 opacity/transform。`[+]` 发布按钮**不做**错位，保持纯净。

### 文件结构

- 左：390×844 的手机外壳，内含约 **50 个 `.page` div**（每个是一个页面/态），靠右侧导航的 `nav()` 与页内 `go()` 函数切换显隐。
- 右：图例面板——页面导航（分组）、文件说明、Tab 图标状态、配色、字体、圆角、QA 验证清单。

## 3. 主要流程串联（页内 `go()` / 导航跳转事实）

**访客 → 登录 → 引导主链：**
P-01 Splash → P-02 访客 Feed →（点登录功能）P-04 登录引导弹窗 / 或 App Bar「Masuk」→ P-05 Google 登录 →（新用户）P-06 昵称 → P-07 宠物状态选择
- 选「有宠物」→ P-08 建档引导 → P-30 建档 → P-31 建档成功
- 选「没有」→ 直接进 P-10 首页
→ 期间 P-09 通知权限请求；老用户登录后回上次所在页。

**AI 分诊链：**
P-17 问诊 →「Mulai Diagnosa」→ P-18 上传 →「Analisis」→ P-19/19b/19c 结果（黄/绿/红）
- 黄/红结果「立即咨询兽医」→ P-21 等待匹配 →（匹配成功）P-22 对话

**兽医问诊链：**
P-17 问诊 / P-20 入口 →「Mulai Konsultasi」→ P-21 等待 → P-22 对话 →「Akhiri」结束 → P-23 评分 → P-25 存档确认（存进档案）

**兽医端工作链：**
V-01b 登录 → V-01/V-02 队列 →「Lihat Detail」→ V-03 案例详情 →「Ambil Kasus」→ V-05 对话 → V-08 最终诊断 → V-06 历史；V-07 我的 / V-ST 状态切换。

**成长档案链：**
P-27 档案 ↔ P-28 日历；P-33 里程碑 → P-33b 详情抽屉 / P-34 图鉴 / P-35 解锁庆祝；P-36 名片对外分享。

## 4. 几个值得记下的设计意图（原型已显式体现）

- **访客可完整浏览 Feed**，仅在触碰核心功能时软引导登录（符合「按需登录控 MAU」的姿态）。
- **登录引导文案说好处、不说「功能不支持」**（P-04 注释明确这么写）。
- **AI 结果三态分级**，红态强制 5 秒阅读倒计时才能关闭——安全攸关的「红色态」设计。
- **删账 = 30 天冷静期 + 键入邮箱二次确认**，呼应注销级联/匿名化的合规要求。
- **问诊记录存档明确「私人、不分享」**（P-25）。
- **页面态覆盖完整**：空态(Feed/通知/时间线)、错误态(Feed/网络)、审核中/被拒都画了——不是只有 happy path。
- **档案锁定字段**：种类、生日不可改（P-32）。
- 全量文案为**印尼语**，目标市场印尼；品牌名 TailTopia。
